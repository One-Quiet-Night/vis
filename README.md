# React-Simple-Maps

## Install
```
npm install --save react-simple-maps
yarn add -save react-simple-maps
```

## 3 components for basic SVG map chart
- ComposableMap
- Geographies
- Geography
- And a valid topojson/geojson file

## More tools
- `ZoomableGroup`
- Tool tip : istall `react-tooltip` to enable a tooltip
